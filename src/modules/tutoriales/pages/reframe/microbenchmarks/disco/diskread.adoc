[#descripcion]
== DescripciГіn

La velocidad de lectura de un disco es el parГЎmetro que determina la rapidez con la que se puede leer un archivo en el sistema. Una baja velocidad de lectura puede ser signo de un disco defectuoso o de una configuraciГіn inadecuada, por lo que es importante conocer el valor de este parГЎmetro.

Existen diferentes programas y utilidades para determinar este valor, sin embargo, muchos de ellos requieren permisos de superusuario (root) para su ejecuciГіn, con el fin de evitar este problema, se optГі por utilizar el comando `dd` del del paquete https://www.gnu.org/software/coreutils/[GNU Core Utilities].


[#ejecucion]
== EjecuciГіn

1. Cargue el mГіdulo de Coreutils:
+
[source, shell]
----
[t.800@yoltla ~]$ module load coreutils-8.32-gcc-7.2.0-43jmyud
----

2. Cree el archivo a leer:
+
[source, shell]
----
[t.800@yoltla ~]$ sync; dd if=/dev/zero of=/tmp/read_file bs=1M count=2048 oflag=dsync status=progress; sync
----

3. Descarte de la memoria cachГ© los datos del archivo a leer:
+
[source, shell]
----
[t.800@yoltla ~]$ dd if=/tmp/read_file iflag=nocache count=0
[t.800@yoltla ~]$ dd of=/tmp/read_file oflag=nocache conv=notrunc,fdatasync count=0
----

4. Ejecute el comando:
+
[source, shell]
----
[t.800@yoltla ~]$ sync; dd if=/tmp/read_file of=/dev/null bs=1M status=progress; sync
----

[#individualmente]
==== *Individualmente*

Para lanzar pruebas de forma individual, ubГӯquese dentro del directorio de la prueba de interГ©s, y ejecute el comando:

[source, shell]
----

----

Por ejemplo, para lanzar la prueba de los nodos NC, ejecute el comando:

[source, shell]
----
[t.800@yoltla nc]$ reframe -c diskread_nc.py -r
----


[#etiquetas]
==== *Etiquetas*

Utilizando etiquetas puede lanzar mГәltiples pruebas con un solo comando. Para lanzar todas las pruebas, siga los siguientes pasos:

. UbГӯquese en el directorio raГӯz _diskread_:
+
[source, shell]
----

----

. Cree el directorio _logs_:
+
[source, shell]
----

----

. Ejecute el comando:
+
[source, shell]
----
[t.800@yoltla diskread]$ reframe -c . -R -t disk -t read -r
----
